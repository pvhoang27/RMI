Kết quả là STRING  → result.getBytes()
	byte[] data = result.getBytes();
	sv.submitData(studentCode, qCode, data);


Kết quả là INT     → String.valueOf(x).getBytes()
	int x = 12345;
	String result = String.valueOf(x);    // "12345"
	byte[] data = result.getBytes();      // thành [49,50,51,52,53]
	sv.submitData(studentCode, qCode, data);
có thể thấy là ta cần phải chuyển int sang string rồi string -> byte

Kết quả là CHAR    → String.valueOf(c).getBytes()
1 chữ
	char c = 'A';
	String result = String.valueOf(c);
	byte[] data = result.getBytes();
	sv.submitData(studentCode, qCode, data);
1 mảng 
	char[] arr = {'A','B','C'};
	String result = new String(arr);  // "ABC"
	byte[] data = result.getBytes();
	sv.submitData(studentCode, qCode, data);
Kết quả là ARRAY ,... ((mảng int, list…))   → ghép thành String → getBytes()
int[] arr = {1, 3, 5};

String result = "";
for (int i = 0; i < arr.length; i++) {
    result += arr[i];         // nối số vào chuỗi
    if (i < arr.length - 1)   // nếu không phải phần tử cuối
        result += ",";        // thì thêm dấu phẩy
}

System.out.println(result);  // "1,3,5"
